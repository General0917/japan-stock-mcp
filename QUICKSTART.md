# クイックスタートガイド

## 5分で始める日本株式分析

### ステップ1: セットアップ（初回のみ）

```bash
cd japan-stock-mcp
./setup.sh
```

### ステップ2: 設定ファイルの編集

#### Claude Desktop の場合

macOSの場合:
```bash
code ~/Library/Application\ Support/Claude/claude_desktop_config.json
```

Windowsの場合:
```cmd
notepad %APPDATA%\Claude\claude_desktop_config.json
```

以下を追加:
```json
{
  "mcpServers": {
    "japan-stock": {
      "command": "node",
      "args": ["/your/path/to/japan-stock-mcp/dist/index.js"]
    }
  }
}
```

#### Cline (VS Code拡張) の場合

VS Codeの設定 (Cmd/Ctrl + ,) → settings.json を開く

```json
{
  "mcp.servers": {
    "japan-stock": {
      "command": "node",
      "args": ["/your/path/to/japan-stock-mcp/dist/index.js"]
    }
  }
}
```

### ステップ3: 再起動

- Claude Desktop: アプリを再起動
- VS Code: ウィンドウをリロード (Cmd/Ctrl + R)

### ステップ4: 使ってみよう！

#### 例1: 単一銘柄の分析

```
トヨタ自動車（7203）の株価を分析してください
```

AIが自動的に以下を実行:
1. 株価データ取得
2. テクニカル指標計算
3. 短期・中期・長期の投資判断

#### 例2: 複数銘柄の比較

```
以下の銘柄を比較して、短期投資に最適なものを教えてください:
- 7203 (トヨタ自動車)
- 6758 (ソニーグループ)
- 9984 (ソフトバンクグループ)
```

#### 例3: セクター分析

```
日本の自動車メーカー主要3社を分析して、
どの会社が今後有望か教えてください:
7203, 7267, 7201
```

#### 例4: 投資期間別の推奨銘柄

```
中期投資（3-6ヶ月）で最も有望な銘柄TOP3を教えてください。
対象銘柄: 7203, 6758, 9984, 8306, 9432, 6861
```

## よくある質問

### Q1: どの銘柄コードを使えばいいですか？

A: `stock-symbols.md` ファイルに主要銘柄の一覧があります。
日本の上場企業なら4桁の数字コードで検索できます。

### Q2: 分析の精度は？

A: テクニカル分析（RSI、MACD、移動平均線など）に基づいています。
ファンダメンタル分析は含まれていないため、あくまで参考情報としてご利用ください。

### Q3: リアルタイムのデータですか？

A: Yahoo Finance APIを使用しており、多少の遅延がある場合があります。

### Q4: 投資アドバイスとして使えますか？

A: 本ツールは情報提供のみを目的としており、投資助言ではありません。
最終的な投資判断はご自身の責任で行ってください。

### Q5: エラーが出ます

A: よくある原因:
- ビルドしていない → `npm run build` を実行
- パスが間違っている → 絶対パスを使用
- Node.jsがインストールされていない → Node.js 18以上をインストール

## トラブルシューティング

### ツールが表示されない

1. 設定ファイルのパスが正しいか確認
2. ビルドが完了しているか確認 (`dist/index.js` が存在するか)
3. アプリケーションを完全に再起動

### 株価データが取得できない

1. インターネット接続を確認
2. 銘柄コードが正しいか確認（4桁の数字）
3. Yahoo Finance APIがアクセス可能か確認

### インスペクターでテスト

```bash
npm run inspector
```

これでMCPサーバーの動作を直接確認できます。

## 高度な使い方

### カスタム分析

ソースコードを編集して独自の分析ロジックを追加できます:
- `src/stock-analyzer.ts`: 分析ロジック
- `src/stock-api.ts`: データ取得
- `src/index.ts`: MCPツール定義

### 開発モード

```bash
npm run watch
```

ファイルを編集すると自動的に再ビルドされます。
