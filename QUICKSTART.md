## 5分で始める日本株式総合分析

### ステップ1: セットアップ（初回のみ）

```bash
cd japan-stock-mcp
./setup.sh
```

### ステップ2: 設定ファイルの編集

#### Claude Desktop の場合

macOSの場合:
```bash
code ~/Library/Application\ Support/Claude/claude_desktop_config.json
```

Windowsの場合:
```cmd
notepad %APPDATA%\Claude\claude_desktop_config.json
```

以下を追加:
```json
{
  "mcpServers": {
    "japan-stock": {
      "command": "node",
      "args": ["/your/path/to/japan-stock-mcp/dist/index.js"]
    }
  }
}
```

#### Cline (VS Code拡張) の場合

VS Codeの設定 (Cmd/Ctrl + ,) → settings.json を開く

```json
{
  "mcp.servers": {
    "japan-stock": {
      "command": "node",
      "args": ["/your/path/to/japan-stock-mcp/dist/index.js"]
    }
  }
}
```

### ステップ3: 再起動

- Claude Desktop: アプリを再起動
- VS Code: ウィンドウをリロード (Cmd/Ctrl + R)

### ステップ4: 使ってみよう！

#### 🌟 例1: 総合分析（テクニカル + 業績）【推奨!】

```
トヨタ自動車（7203）を業績も含めて総合的に分析してください
```

AIが自動的に以下を実行:
1. 株価データ取得（テクニカル分析用）
2. 財務データ取得（PER、PBR、ROE、売上成長率など）
3. テクニカル指標計算（RSI、MACD、移動平均線）
4. ファンダメンタル評価（バリュエーション、収益性、成長性、財務健全性）
5. 短期・中期・長期の総合投資判断

**出力される情報:**
- 企業基本情報（セクター、業種、従業員数）
- ファンダメンタルスコア（4軸評価）
- テクニカル指標
- 総合投資判断（期間別）
- 具体的な推奨内容

#### 例2: 財務データの確認

```
ソニーグループ（6758）の財務状況を詳しく教えてください
```

**取得できるデータ:**
- バリュエーション指標（PER、PBR、EPS）
- 収益性指標（ROE、配当利回り）
- 成長性指標（売上・利益成長率）
- 財務健全性（負債比率、流動比率）

#### 例3: 複数銘柄の総合比較

```
以下の銘柄を業績も含めて比較分析してください:
- 7203（トヨタ自動車）
- 6758（ソニーグループ）  
- 9984（ソフトバンクグループ）
```

**比較項目:**
- 短期・中期・長期の総合スコア
- テクニカル評価 × ファンダメンタル評価
- 各銘柄の強みと弱み
- 投資期間別のランキング

#### 例4: 長期投資向けの銘柄選定

```
以下の銘柄から、長期投資（5年以上）に最適なものを
業績面も考慮して教えてください:
7203, 6758, 9984, 8306, 9432
```

長期投資では:
- ファンダメンタルが70%のウェイト
- 成長性と財務健全性を重視
- ROE、売上成長率、負債比率などを総合評価

#### 例5: テクニカル分析のみ（従来機能）

```
任天堂（7974）の株価チャートを分析してください
```

テクニカル指標のみで短期・中期・長期の判断を提供

## 分析手法の違い

### テクニカル分析のみ（従来）
- 株価チャートとテクニカル指標のみ
- 短期トレードに適している
- データ取得が高速

### 総合分析（NEW!）
- テクニカル + ファンダメンタルを統合
- **短期**: テクニカル70% + ファンダメンタル30%
- **中期**: テクニカル50% + ファンダメンタル50%
- **長期**: テクニカル30% + ファンダメンタル70%
- より総合的な投資判断が可能

## よくある質問

### Q1: どの分析方法を使えばいいですか？

A: 投資期間に応じて選択してください:
- **デイトレード・短期**: テクニカル分析のみでOK
- **数週間〜数ヶ月**: 総合分析を推奨
- **長期投資**: 総合分析を強く推奨（ファンダメンタル重視）

### Q2: PER、PBR、ROEって何ですか？

A: 
- **PER（株価収益率）**: 株価が1株利益の何倍か。低いほど割安。
- **PBR（株価純資産倍率）**: 株価が1株純資産の何倍か。1倍以下は割安。
- **ROE（自己資本利益率）**: 資本をどれだけ効率的に使って利益を生んでいるか。15%以上が優秀。

ツールが自動的に評価してくれるので、詳しく知らなくても大丈夫です。

### Q3: 財務データが取得できない銘柄もありますか？

A: はい。小型株や一部の銘柄では財務データが不完全な場合があります。
その場合でもテクニカル分析は実行されます。

### Q4: どの銘柄コードを使えばいいですか？

A: `stock-symbols.md` ファイルに主要銘柄の一覧があります。
日本の上場企業なら4桁の数字コードで検索できます。

### Q5: リアルタイムのデータですか？

A: Yahoo Finance APIを使用しており、多少の遅延がある場合があります。

### Q6: 投資アドバイスとして使えますか？

A: 本ツールは情報提供のみを目的としており、投資助言ではありません。
最終的な投資判断はご自身の責任で行ってください。

## トラブルシューティング

### ツールが表示されない

1. 設定ファイルのパスが正しいか確認
2. ビルドが完了しているか確認 (`dist/index.js` が存在するか)
3. アプリケーションを完全に再起動

### 財務データが取得できない

1. インターネット接続を確認
2. 銘柄コードが正しいか確認（4桁の数字）
3. Yahoo Finance APIがアクセス可能か確認
4. 小型株の場合、データが不完全な可能性あり

### エラーが出る

```bash
# node_modulesを削除して再インストール
rm -rf node_modules package-lock.json
npm install
npm run build
```

### インスペクターでテスト

```bash
npm run inspector
```

これでMCPサーバーの動作を直接確認できます。

## 高度な使い方

### 1. セクター全体の分析

```
自動車セクターの主要企業を総合分析してください:
7203（トヨタ）, 7267（ホンダ）, 7201（日産）

業績面での成長性と株価の割安性を比較して、
最も投資価値が高い銘柄を教えてください。
```

### 2. 配当重視の投資

```
以下の銘柄から、配当利回りが高く財務も健全な
銘柄をランキングしてください:
8306, 8316, 8411, 9432, 9434
```

### 3. 成長株の発掘

```
以下の銘柄の中で、売上・利益成長率が高く、
かつROEも優秀な成長株を見つけてください:
6758, 4063, 6861, 6098, 4689
```

### 4. バリュー株の発掘

```
PERとPBRが低く割安で、かつ財務健全性も高い
バリュー株を見つけてください:
候補: 7203, 8058, 8031, 5401, 5711
```

## カスタマイズ

分析ロジックをカスタマイズする場合:
- `src/stock-analyzer.ts`: テクニカル分析ロジック
- `src/financial-api.ts`: 財務データ取得
- `src/comprehensive-analyzer.ts`: 総合分析ロジック
- `src/index.ts`: MCPツール定義

編集後:
```bash
npm run build
```

開発モード（自動再ビルド）:
```bash
npm run watch
```
