# 使用例 - 日本株式分析MCPサーバー v2.0

## 基本的な使い方

### 1. テクニカル分析のみ

#### 単一銘柄のテクニカル分析
```
トヨタ自動車（7203）をテクニカル分析してください
```

**結果に含まれるもの:**
- 短期・中期・長期の投資判断
- RSI、MACD、移動平均線などのテクニカル指標
- 各期間の推奨シグナル（BUY/SELL/HOLD）

#### 現在価格の確認
```
ソニーグループ（6758）の現在価格を教えてください
```

---

## ファンダメンタルズ分析（NEW!）

### 2. 企業業績の分析

#### 財務データの取得と分析
```
ソフトバンクグループ（9984）のファンダメンタルズを分析してください
```

**結果に含まれるもの:**
- **バリュエーション評価**: PER、PBR、配当利回り
- **財務健全性評価**: ROE、負債比率、流動比率
- **収益性評価**: 営業利益率、純利益率、EPS
- 各項目のスコア（0-100）と評価理由

#### より詳しい質問例
```
任天堂（7974）は割安ですか？PERとPBRから判断してください
```

```
キーエンス（6861）の財務健全性を評価してください
```

```
三菱UFJ（8306）の配当利回りはどうですか？
```

## 総合分析（テクニカル × ファンダメンタルズ）

### 3. 統合分析

#### 完全な総合分析
```
トヨタ自動車（7203）を総合的に分析してください
```

**結果に含まれるもの:**
- テクニカル分析結果（短期・中期・長期）
- ファンダメンタルズ分析結果（バリュエーション、財務健全性、収益性）
- **総合スコア**（0-100）
- **総合推奨レベル**（STRONG_BUY/BUY/HOLD/SELL/STRONG_SELL）
- **投資サマリー**: 主要なポイントを要約

#### セクター別の総合分析
```
自動車セクターの主要3社を総合的に分析して比較してください:
- 7203（トヨタ自動車）
- 7267（本田技研工業）
- 7201（日産自動車）

テクニカルとファンダメンタルズの両面から、どの企業が最も有望か教えてください。
```

## 投資期間別の推奨

### 6. 短期投資（1ヶ月以内）

```
短期投資で有望な銘柄TOP3を教えてください。
候補: 7203, 6758, 9984, 8306, 9432, 6861
```

### 7. 中期投資（3-6ヶ月）

```
中期投資で最も有望な銘柄を教えてください。
候補銘柄: トヨタ、ソニー、ソフトバンク、任天堂、キーエンス
```

### 8. 長期投資（6ヶ月以上）

```
長期投資に適した銘柄TOP5を、ファンダメンタルズ重視でランキングしてください。
候補: 7203, 6758, 9984, 7974, 6861, 4063, 8306
```

## トラブルシューティング

### データが取得できない場合

```
7203の財務データが取得できません。どうすればいいですか？
```

### 分析結果の解釈

```
「PER 15.2」という数値は高いですか、低いですか？
業種平均と比較して評価してください。
```

```
「ROE 12.5%」という数値の意味を教えてください。
これは良い水準ですか？
```

## ヒント

1. **総合分析を優先**: テクニカルとファンダメンタルズの両面から分析する`analyze_comprehensive`を使用すると、より信頼性の高い判断ができます。

2. **複数銘柄の比較**: 単一銘柄だけでなく、複数銘柄を比較することで相対的な魅力度が明確になります。

3. **投資期間を明確に**: 短期・中期・長期で評価基準が異なるため、自分の投資スタイルに合った期間で評価してください。

4. **財務指標の理解**: PER、PBR、ROEなどの意味が分からない場合は、MCPに質問すれば説明してくれます。

5. **定期的な再評価**: 株価や業績は常に変動するため、定期的に再分析することをお勧めします。
